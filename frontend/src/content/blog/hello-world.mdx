# Hello World

Welcome to **ShockStack**, a convention-first monorepo template built with Astro 5, Vue 3, Tailwind 4, and an optional .NET 10 backend.

## What's included

- Design token system powered by Style Dictionary
- Six built-in themes (Dracula, Alucard, Nord, Gruvbox, Midnight, Dawn)
- Header theme picker with persisted user preference
- Content collections for blog, docs, and changelog
- Full CI/CD with GitHub Actions
- Cloudflare Workers or Docker deployment
